Event Apps Latest Trend in Event Industry?

Its a new term in event industry - in use for the last 2 years only and becoming popular right now, as you read this website. It is a type of app generally available for all the possible devices like iPhone, Android, Webetc, that event attendees download for the specific event. It is filled with unique event information like agendas or speakers profiles and can also serve as a tool to connect and chat with other attendees, get important updates from event organisers and other functions too.
